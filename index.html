Hermes Agent as a skill file: Upgrade your existing agent, no need to migrate.

/hermes-agent-as-a-skill-file: A closed self-improvement loop that makes your agent smarter with every task it completes.

Speedrun AI Labs · Inspired by Nous Research · v1.0 · April 2026 · MIT License

Nous Research released Hermes Agent -- a genuinely useful learning loop for autonomous agents. This skill packages those patterns as a portable SKILL.md file: auto skill creation, cross-session recall, memory capacity management, periodic self-checks, compound learning, skill versioning, and inter-agent skill sharing. Drop it into your existing agent. No migration. No new runtime.

Every AI edits files without reading them first. This skill fixes that.

/read-before-write: Force any AI agent to read the full file before changing a single line — like a contract lawyer reads the whole document before touching it.

Speedrun AI Labs · v1.0 · April 2026 · MIT License

On the morning this skill was written, two AI systems — our in-house agent and Claude.ai — made the same mistake within an hour of each other: both overwrote existing files without reading them first. Silently. No warning. This skill hardens any AI agent against that failure: mandatory full-file read before any edit, patch-not-replace discipline, conflict flagging before proceeding, and a hard stop if the file can't be read. Built in Sydney by Speedrun AI Labs — where we caught this bug the hard way.

AI writes your code. This skill audits it like an accountant: evidence first, findings second, fixes only when it can prove them.

/deep-audit: A four-phase code auditor with operating modes, severity definitions, and an evidence standard that prevents overclaiming.

Speedrun AI Labs · v1.1 · March 2026 · MIT License

Catches what pattern-matching tools miss: business logic flaws, race conditions, silent failures, subtle auth bypasses, and the kind of bugs that survive years of human review.
